# **App Name**: Aura Stories

## Core Features:

- Intelligent Discovery Hub: A dynamic Flutter dashboard featuring aesthetic carousel banners and genre-based filters like Romantik and Fantastik for quick story exploration.
- Smart Recommendations Engine: An AI-powered tool that analyzes reading patterns to populate the 'Zevkinize Göre' section with personalized book suggestions.
- Persistent Mobile Navigation: A 4-tab BottomNavigationBar (Kitaplık, Keşfet, Ödüller, Ben) built for seamless view transitions in the Flutter environment.
- Interactive Story Catalog: Richly styled cards for popular books including vertical covers and metadata, utilizing a modular widget architecture.
- Personal Library Tracker: A private space for users to save their reading progress and track favorite stories, persisted via a cloud database.
- Rewards & Engagement Dashboard: A dedicated mobile interface to track earned platform bonuses and visual achievements in a modern card format.
- Content Synchronization Service: A backend integration that serves high-quality Turkish book data and aesthetic image pointers to the Flutter client.

## Style Guidelines:

- Primary Color: Royal Amethyst (#9B67D4) - a soft, creative purple used for main UI elements.
- Background Color: Morning Mist Purple (#F8F4FA) - a heavily desaturated version of the primary to create a premium, clean reading environment.
- Accent Color: Deep Iris (#4D53B2) - an analogous indigo providing sharp contrast for action buttons and high-priority filters.
- Headline Font: 'Playfair Display' (Serif) - high contrast and geometric for an elegant, literary feel.
- Body Font: 'PT Sans' (Sans-Serif) - humanist and warm for optimal legibility during long reading sessions.
- Modular grid layout with a top carousel, followed by a horizontal pill-shaped chip row and vertical list items.
- Minimalist outline icons for navigation (Book, Explore, Card, Person) with slight glow effects when active.
- Smooth parallax transitions for book covers and horizontal 'snap' scrolling for the category chips.
